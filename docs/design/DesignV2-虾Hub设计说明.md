# 虾Hub V2 设计说明

> Version 2.0 · 2026-06-15
> 关键词：紧凑 · 冷色科技感 · 精致克制

---

## 一、设计理念

V1 走的是"暖色暗调 + 圆润亲和"路线（珊瑚色 #C27C68、16px 大圆角、24px 页面边距、厚卡片），适合展示 Demo 但日常使用显得松散。V2 的核心转变是 **从"展示型"转向"工具型"**——让用户在一屏内看到更多信息，操作更精准，视觉更克制。

三条设计原则：

**密度优先（Density）**：缩紧间距、减小字号、压缩卡片高度，一屏展示更多内容。不靠减少功能来紧凑，而是让每个元素占更少空间。

**边框替代填充（Border over Fill）**：V1 大量使用 surface 层叠（bg → surface → surface2 → surface3）来区分层级，占面积且显臃肿。V2 用 1px hairline 边框 + 微弱色差代替厚实的填充色块，视觉上更轻盈、更精确。

**冷色克制（Cool Restraint）**：从暖棕色调全面转向冷蓝灰色调。品牌色从珊瑚换为宝石蓝，辅以紫罗兰作次级强调。整体色温降低，传达"精准工具"而非"温暖陪伴"的气质。

## 二、色彩体系

### 2.1 背景与表面色（Cool-Toned Dark）

从暖灰（#111110 带黄底）转向冷灰（#08090D 带蓝底）。表面层级不再靠大幅明度跳跃来区分，而是用 hairline 边框 + 微弱色差。

| Token | V1 | V2 | 变化说明 |
|:---|:---|:---|:---|
| `--bg` | `#111110` | `#08090D` | 底色变冷，带微弱蓝调 |
| `--surface` | `#1A1917` | `#0E1016` | 一级表面，与 bg 差值缩小 |
| `--surface2` | `#232220` | `#15171E` | 二级表面 |
| `--surface3` | `#2C2B28` | `#1C1F28` | 三级表面 |
| `--surface-elevated` | `#1F1E1C` | `#12141B` | 浮层专用 |
| `--border` | (无) | `rgba(255,255,255,0.06)` | **新增** hairline 边框色，V2 的核心视觉分隔手段 |
| `--border-accent` | (无) | `rgba(79,131,255,0.30)` | **新增** 强调边框（选中态、激活态） |

**层级策略变化**：

```
V1: bg ← 大色差 → surface ← 大色差 → surface2 ← 大色差 → surface3
     (依赖明度差分层级)

V2: bg ← hairline border → surface ← hairline border → surface2
     (依赖精细边框分层级，色差仅作辅助)
```

### 2.2 文字色（Cool White Tones）

文字基底从暖白（rgba(245,244,240)）转向冷白（rgba(235,239,250)），带微弱蓝调。

| Token | V2 值 | Alpha | 用途 |
|:---|:---|:---|:---|
| `--text-1` | `#EBEFFA` | 100% | 标题、正文、重要数据 |
| `--text-2` | `rgba(235,239,250,0.55)` | 55% | 辅助说明、次要标签 |
| `--text-3` | `rgba(235,239,250,0.30)` | 30% | 时间戳、预览文本、占位符 |
| `--text-4` | `rgba(235,239,250,0.14)` | 14% | 禁用态、极弱装饰 |

### 2.3 品牌色（Sapphire + Violet）

品牌主色从珊瑚色换为**宝石蓝**，次级强调色为**紫罗兰**。蓝色传达精准、可信、科技感；紫色增加高端辨识度，避免与市面上蓝色 App 撞色。

| Token | V1 | V2 | 说明 |
|:---|:---|:---|:---|
| `--accent` | `#C27C68` 珊瑚 | `#4F83FF` 宝石蓝 | 主 CTA、活跃 Tab、选中态 |
| `--accent-hover` | `#D08E7C` | `#6B9AFF` | hover/pressed 态提亮 |
| `--accent-muted` | `rgba(194,124,104,0.12)` | `rgba(79,131,255,0.10)` | 轻量背景（banner、标签底色） |
| `--accent-glow` | `rgba(194,124,104,0.18)` | `rgba(79,131,255,0.20)` | 按钮发光阴影 |
| `--accent2` | (无) | `#9B7AFF` 紫罗兰 | **新增** 次级强调色（成就、特殊标记） |
| `--accent2-muted` | (无) | `rgba(155,122,255,0.10)` | 紫罗兰轻量背景 |
| `--gold` | (无) | `#E8C574` | **新增** 金色高光（里程碑、成就徽章） |

**品牌色使用规则**：

- `--accent`（蓝）用于：活跃导航项、主按钮、发送按钮、未读徽章、选中态描边、链接文字
- `--accent2`（紫）用于：成就系统图标、特殊标记、Agent 在线脉冲动画（与蓝色形成双色调）
- `--gold` 用于：里程碑庆祝动画、成就解锁高亮、成长面板星级
- 三种强调色不可同时出现在同一组件上，保持视觉清晰

## 六、组件级布局变化

### 6.1 首页 — 虾列表（核心变化最大的页面）

#### Header 区域

V1 是 30px 大标题 + 统计栏三张等大卡片，占掉将近 1/3 屏幕。V2 大幅压缩：

```
V1 Header (约 180px):
┌─────────────────────────┐
│  [≡]  虾Hub         [+] │  ← 54px (含 status bar)
│                         │
│ ┌──────┐┌──────┐┌──────┐│  ← 统计栏 ~100px
│ │活跃 2││在线 5││消息 42││    三张等大卡片
│ │ /3   ││ /8   ││      ││
│ └──────┘└──────┘└──────┘│
└─────────────────────────┘

V2 Header (约 100px):
┌─────────────────────────┐
│ 虾Hub    2/3 ● 5/8 ● 42│  ← 标题行内联统计
│  ─────────────────────  │    hairline 底部分隔
└─────────────────────────┘
```

**关键变化**：

- 标题 "虾Hub" 缩小为 24px，与统计数字在同一行内联展示
- 统计数据从三张大卡片变为**行内联排小指标**：`2/3` (活跃实例) + 绿点 + `5/8` (在线虾) + 绿点 + `42` (消息数)
- 每个指标间用 `·` 或 `|` 分隔，字号 13px，数字 tabular-nums
- Header 总高度从 ~180px 压缩到 ~56px，**节省约 70% 的头部空间**

#### 分组头

```
V1:
┌───────────────────────────┐
│ ● 我的云服务器         3 ▼│  ← padding 16px 24px
└───────────────────────────┘

V2:
  ● 我的云服务器 · 3只               ▼
  ───────────────────────────────────
```

- 改为**内联文字行**：状态点 + 实例名 + `·` + 数量 + `只`，右对齐 chevron
- padding 从 `16px 24px` 压缩到 `10px 16px`，min-height 保持 44px（触控目标）
- 下方加一条 hairline 分隔线（`--border`），替代 surface 填充色
- Press 态保持 V1 修正后的 `opacity: 0.5`（已在 ComponentSpec 中定义）

| `--ease` | `cubic-bezier(0.16, 1, 0.3, 1)` | **不变** | expo-out，保持 |
| `--ease-spring` | `cubic-bezier(0.34, 1.56, 0.64, 1)` | **移除** | V2 不用弹性超调，更克制 |

### 8.3 按压反馈

保持 V1 修正后的方案（scale + opacity），但 scale 值进一步缩小：

| 组件 | V1 Scale | V2 Scale | 说明 |
|:---|:---|:---|:---|
| Agent 卡片 | 0.98 | **0.97** | |
| 发送按钮 | 0.92 | **0.90** | |
| 主按钮 | 0.97 | **0.95** | |
| 分组头 | 不缩放 + opacity 0.5 | **不变** | |

## 九、V1 → V2 视觉差异总结

| 维度 | V1 | V2 | 气质差异 |
|:---|:---|:---|:---|
| 色温 | 暖（棕底 + 珊瑚） | 冷（蓝底 + 宝石蓝） | 温润亲和 → 精准科技 |
| 密度 | 宽松（24px padding, 84px 卡片） | 紧凑（16px padding, 60px 卡片） | 展示型 → 工具型 |
| 层级手段 | surface 层叠填充色 | hairline border + 微弱色差 | 厚重 → 轻盈 |
| 圆角 | 大（16-20px） | 小（10-14px） | 圆润可爱 → 精密精确 |
| 阴影 | 多层阴影 | 极少阴影，改用 border | 浮起感 → 贴合感 |
| 动效速度 | 偏慢（200-500ms） | 偏快（150-400ms） | 从容 → 敏捷 |
| 强调色系 | 单色（珊瑚） | 双色（蓝 + 紫） | 统一 → 层次 |
| 统计展示 | 三张大卡片 | 行内联排指标 | 占空间 → 信息流 |
| 导航指示器 | 填充色胶囊 | 2px 底部线条 | 厚实 → 极简 |
| 卡片分隔 | 间距 + 背景色 | hairline border + 极小间距 | 留白分隔 → 线条分隔 |
